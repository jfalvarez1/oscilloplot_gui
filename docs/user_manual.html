Introduction

Oscilloplot is a powerful Python GUI application for generating XY audio signals that can be displayed on a real analog oscilloscope. Create mesmerizing visual patterns, Lissajous figures, spirals, and custom drawings that come to life on your oscilloscope's phosphor screen!

Key Features

Real-Time Preview

Classic CRT green phosphor display with density-based rendering shows your patterns in real-time.

Audio Output

Generate stereo audio signals (Left = X, Right = Y) for real oscilloscope display.

Multiple Generators

Sum of Harmonics, Archimedean Spirals, Random Harmonics, Freehand Drawing, and more.

15+ Effects

Rotation, fade, shrink, wavy, tremolo, ring mod, echo, kaleidoscope, distortion, and more.

Image Converter

Convert photos and drawings to oscilloscope-ready coordinates.

WAV Export

Save your generated audio as stereo WAV files for later use.

Installation

Prerequisites

  • Python 3.8 or higher
  • A stereo audio output device
  • (Optional) An analog oscilloscope in XY mode

Install Dependencies

Using the requirements file:

pip install -r requirements.txt

Or install manually:

pip install numpy>=1.20.0 sounddevice>=0.4.0 matplotlib>=3.3.0 soundfile>=0.10.0

For the image converter, you'll also need:

pip install opencv-python
Make sure you have a working audio output device before running the application. The application uses the system's default audio output.

Quick Start Guide

Standard Version

python oscilloscope_gui.py

Optimized Version (Recommended)

python oscilloscope_gui_optimized.py

Getting Started in 5 Steps

  1. Launch the application - Run either version of the GUI
  2. Load or generate a pattern - Click "Generate Test Pattern" for a quick demo, or try "Sum of Harmonics" for Lissajous figures
  3. Adjust parameters - Use the control panel on the left to modify audio settings and effects
  4. Click "Apply & Generate" - This processes your pattern with all selected effects
  5. Click "Play Audio" - Listen to the output and watch the live preview
For real oscilloscope display, connect your computer's stereo audio output to your scope's X and Y inputs.

Audio Parameters

These parameters control how your pattern is converted to audio:

Base Sample Rate (Hz)
Range: 100 - 10,000 Hz | Default: 1,000 Hz

The base sampling rate for the pattern. Higher values create more detailed patterns but result in larger audio files. This value is multiplied by the Playback Multiplier to get the actual audio sample rate.

Playback Multiplier
Range: 10 - 500 | Default: 100

Multiplies the base rate for actual audio output. The formula is:

Actual Rate = Base Sample Rate × Playback Multiplier

For example: 1,000 Hz × 100 = 100,000 Hz (100 kHz)

Actual Rate

This read-only label shows the calculated actual sample rate based on your Base Sample Rate and Playback Multiplier settings.

Duration (seconds)
Range: 5 - 120 seconds | Default: 15 seconds

Total playback duration of the generated audio. Longer durations create larger files but allow effects like rotation to complete more cycles.

Pattern Repeats
Range: 1 - 2,000 | Default: 200

How many times the pattern repeats during playback. More repeats create smoother animated effects like rotation, fade sequences, and shrink effects. The rotation info label shows how many full rotations will occur based on this value.

The "Actual Rate" label updates automatically when you change the Base Sample Rate or Playback Multiplier.

Pattern Generators - Detailed Guide

Sum of Harmonics (Lissajous Figures)

Create beautiful Lissajous figures by combining multiple sine waves. This is the mathematical foundation of many classic oscilloscope patterns.

Mathematical Basis

X = A1·sin(f1·t + φ1) + A2·sin(f2·t + φ2) + ...
Y = B1·sin(g1·t + ψ1) + B2·sin(g2·t + ψ2) + ...

Harmonic Parameters

ParameterDescription
Amplitude (A, B)Wave height - controls the size of the contribution
Frequency (f, g)Oscillation speed - integer ratios create closed figures
Phase (φ, ψ)Starting position offset in radians

Classic Lissajous Ratios

X:Y RatioShape
1:1Circle, Ellipse, or Line (depends on phase)
1:2Figure-8 (infinity symbol)
2:3Pretzel or bow-tie shape
3:4Complex interlocking loops
3:5Even more complex patterns
Integer frequency ratios (like 1:2, 2:3, 3:4) create closed figures that loop smoothly. Non-integer ratios create patterns that never close, filling the display area over time.

Archimedean Spiral

Generate spiral patterns using polar coordinates converted to Cartesian for display.

Mathematical Basis

r = a + b · θ

Where:

  • r = distance from center (radius)
  • θ = angle in radians
  • a = starting radius (inner size)
  • b = growth rate (spacing between loops)

Cartesian Conversion

X = r · cos(θ)
Y = r · sin(θ)

Parameters

ParameterDescription
a (Start Radius)Inner size of the spiral. 0 starts at center.
b (Growth Rate)Spacing between loops. Higher = wider spacing.
TurnsNumber of complete spiral rotations.

Sound Pad Sequencer

A 16-step pattern sequencer for creating rhythmic, stepping patterns.

Features

Position OffsetShifts the entire pattern position

The Sound Pad is great for creating geometric patterns that step through discrete positions, similar to early vector game graphics.

Sum of Harmonics - Complete Guide

The Sum of Harmonics generator is one of the most powerful pattern creation tools in Oscilloplot. It creates Lissajous figures and complex waveforms by mathematically combining multiple sine and cosine waves. Understanding how each parameter affects the output will help you design stunning visual patterns.

Understanding the Mathematics

Each channel (X and Y) is the sum of one or more harmonic terms. The general formula is:

X(t) = A1·func(f1·t + φ1) + A2·func(f2·t + φ2) + ...
Y(t) = B1·func(g1·t + ψ1) + B2·func(g2·t + ψ2) + ...

Where func is either sin or cos, and t ranges from 0 to 2π.

Parameter Reference

ParameterSymbolEffect on Shape
Function Type sin / cos Determines wave shape. cos is equivalent to sin with a 90° phase shift.
Amplitude A, B Controls the size of that term's contribution. Larger values = bigger influence on the overall shape.
Frequency f, g (ω) Controls how many complete cycles occur. The ratio between X and Y frequencies determines the shape type.
Phase φ, ψ Starting position of the wave (in radians). Shifts the wave left/right in time, changing the figure's orientation.

Shapes Created by Frequency Ratios

The ratio of X frequency to Y frequency is the most important factor in determining the shape. When both channels have a single term, here's what you get:

1:1 Ratio (Equal Frequencies)

Phase DifferenceResulting Shape
0° (0 radians)Diagonal Line (45°) - X and Y are in sync
45° (π/4)Ellipse tilted at an angle
90° (π/2)Circle - X leads Y by quarter cycle
180° (π)Diagonal Line (opposite direction, -45°)
270° (3π/2)Circle (opposite rotation direction)
To create a perfect circle: X = sin(t), Y = cos(t) - which is sin(t) with π/2 phase offset.

Simple Integer Ratios

X:Y RatioShape DescriptionNumber of Lobes
1:2Figure-8 / Infinity symbol (∞)2 lobes horizontally
2:1Figure-8 rotated 90°2 lobes vertically
1:3Trefoil / Three-leaf pattern3 lobes
2:3Pretzel / Bow-tie shapeComplex 6-crossing pattern
3:4Complex interlocking loops12 crossings
3:5Very complex star-like pattern15 crossings
4:5Dense, flower-like pattern20 crossings
Pattern Rule: For ratio a:b, the figure will have (a×b) crossing points when the ratio is in lowest terms. Integer ratios always create closed, repeating figures.

Non-Integer Ratios

When frequencies are not integer ratios (e.g., 1:1.5, 1:π), the pattern never closes. Over time, the beam traces out more and more of the display area, creating filled-in shapes. These are called open Lissajous curves.

How Each Parameter Changes the Shape

Changing Amplitude

Increasing X Amplitude

Stretches the pattern horizontally. Higher values = wider shape.

Increasing Y Amplitude

Stretches the pattern vertically. Higher values = taller shape.

Unequal Amplitudes

Creates elliptical distortion. A 1:1 ratio with unequal amplitudes turns a circle into an ellipse.

Changing Frequency

Increasing X Frequency

Adds more horizontal oscillations. Creates more "lobes" on the left and right sides.

Increasing Y Frequency

Adds more vertical oscillations. Creates more "lobes" on the top and bottom.

Higher Total Frequencies

Creates denser, more intricate patterns with many crossings and loops.

Changing Phase

Small Phase Changes (0-π/4)

Tilts or rotates the figure slightly. Turns lines into ellipses.

π/2 Phase Difference

Maximum "openness" - creates the most rounded version of any pattern (circles, full figure-8s).

π Phase Difference

Inverts one axis - mirrors the pattern.

Adding Multiple Harmonics

The real power comes from combining multiple terms. Each additional harmonic adds complexity and detail to the pattern:

Example: Creating a Star Shape

To create a 5-pointed star:

  • X Channel: sin(5t) with amplitude 1.0
  • Y Channel: cos(5t) with amplitude 1.0
  • Then add smaller harmonics to sharpen the points

Example: Complex Flower Pattern

Combine a base frequency with higher harmonics:

  • X Term 1: sin(3t), amplitude 1.0
  • X Term 2: sin(7t), amplitude 0.3
  • Y Term 1: cos(5t), amplitude 1.0
  • Y Term 2: cos(11t), amplitude 0.2

The higher harmonics add fine detail to the main pattern.

Tips for Complex Patterns

  • Use prime number frequencies - Creates more complex, non-repeating subpatterns
  • Decrease amplitude with frequency - Higher harmonics should have smaller amplitudes for natural-looking shapes
  • Vary phases randomly - Prevents all harmonics from aligning too symmetrically
  • Mix sin and cos - Equivalent to phase shifts, but easier to conceptualize

Animation Features

Phase Sweep

Enabling "Phase Sweep" on a term animates its phase from a start value to an end value over a specified number of steps. This creates:

  • Rotating patterns - The figure appears to spin or tumble
  • Morphing shapes - Smooth transitions between different configurations
  • Pulsing effects - Rhythmic expansion and contraction
For a smoothly rotating circle, sweep the phase from 0 to 2π over many steps.

Frequency Sweep

Sweeping frequency creates dramatic transformation effects:

  • Pattern morphs from one shape to another (e.g., circle → figure-8 → trefoil)
  • Creates "warping" or "breathing" animations
  • Useful for creating transition effects

Classic Recipes

PatternX SettingsY Settings
Perfect Circle sin(1·t), A=1 cos(1·t), B=1
Figure-8 sin(1·t), A=1 sin(2·t), B=1
Butterfly sin(2·t), A=1 sin(3·t), B=1
5-Petal Flower sin(5·t), A=1 cos(4·t), B=1
Spiral Star sin(5·t) + 0.5·sin(2·t) cos(5·t) + 0.5·cos(2·t)
Complex Web sin(7·t) + 0.3·sin(3·t) cos(5·t) + 0.3·cos(11·t)

Archimedean Spiral - Complete Guide

The Archimedean Spiral generator creates classic spiral patterns that expand outward from a center point. Named after the ancient Greek mathematician Archimedes, this spiral has uniform spacing between successive turns, making it visually pleasing and mathematically elegant.

Understanding the Mathematics

In Oscilloplot, the Archimedean spiral is generated using the parametric equations:

X(t) = a · t · sin(b · t)
Y(t) = a · t · cos(b · t)

Where:

  • t = time parameter (ranges from 0 to 4π, giving approximately 2 full rotations)
  • a = amplitude/size parameter (controls overall spiral size)
  • b = frequency parameter (controls how tightly wound the spiral is)

The classic polar form is r = a + b·θ, but Oscilloplot uses the parametric form above which produces similar visual results with independent X and Y control.

Parameter Reference

ParameterRangeEffect on Spiral
a (Amplitude) 0.1 - 5.0 Controls the overall size and how quickly the spiral expands. Higher values = faster growth, larger spiral.
b (Frequency) 0.1 - 10.0 Controls the rotation speed / number of turns. Higher values = more tightly wound spiral with more rotations.

Shapes Created by Parameter Combinations

Effect of Parameter 'a' (Amplitude)

Low a (0.1 - 0.5)

Small, compact spiral. The arms stay close to the center. Good for detailed, concentrated patterns.

Medium a (0.5 - 1.5)

Balanced spiral that uses most of the display area. Best for general use.

High a (1.5 - 5.0)

Large, expansive spiral. Arms extend far from center. May extend beyond display boundaries.

Effect of Parameter 'b' (Frequency)

Low b (0.1 - 0.5)

Very slow rotation. Creates a "lazy" spiral with few turns. Almost looks like a curved line.

Medium b (1.0 - 3.0)

Classic spiral appearance with 2-6 visible turns. Clear spiral structure.

High b (3.0 - 10.0)

Tightly wound spiral with many rotations. Creates dense, spring-like patterns.

Common Spiral Shapes

ShapeX: aX: bY: aY: bDescription
Classic Spiral 1.0 1.0 1.0 1.0 Perfectly symmetric Archimedean spiral
Tight Coil 0.5 5.0 0.5 5.0 Many tight loops, spring-like appearance
Loose Whorl 2.0 0.5 2.0 0.5 Few, widely-spaced turns
Horizontal Elliptical 1.5 1.0 0.5 1.0 Spiral stretched horizontally
Vertical Elliptical 0.5 1.0 1.5 1.0 Spiral stretched vertically
Complex Lissajous-Spiral 1.0 2.0 1.0 3.0 Hybrid pattern with Lissajous-like features

Different X and Y Parameters

When X and Y have different parameter values, you create distorted and hybrid patterns:

Unequal Amplitudes (aX ≠ aY)

  • Creates an elliptical spiral instead of circular
  • If aX > aY: Spiral is wider than tall
  • If aX < aY: Spiral is taller than wide

Unequal Frequencies (bX ≠ bY)

  • Creates complex, Lissajous-like spiral hybrids
  • The spiral develops additional lobes and crossings
  • Integer ratios create repeating patterns; non-integer ratios create filled patterns
Try bX=2 and bY=3 for a fascinating spiral that weaves in and out of itself.

Frequency Sweep

The frequency sweep feature animates the 'b' parameter from a start value to an end value, creating dramatic morphing effects:

Start Frequency → End Frequency

The spiral smoothly transitions from loosely wound to tightly wound (or vice versa). Creates a "breathing" or "pulsing" effect.

Sweep Examples

StartEndEffect
0.5 5.0 Spiral winds tighter over time (coiling in)
5.0 0.5 Spiral unwinds over time (expanding out)
1.0 10.0 Dramatic tightening, spring compression effect
Frequency sweep uses a chirp signal, smoothly transitioning through all intermediate values rather than stepping discretely.

Chaser Effect

The Chaser Effect creates an animated point that traces along the spiral path, similar to a light chasing around a display. This is achieved by showing only a portion of the spiral at a time and sliding that window along the path.

Chaser Parameters

ParameterRangeEffect
Trail Length 5 - 500 points How many points are visible at once. Short = fast-moving dot, long = comet-like trail.
Point Repetitions 1 - 100 How many times each window position is repeated. Higher = slower movement, smoother animation.

Chaser Effect Tips

  • Short trail (5-20) - Creates a single moving point, like a bouncing ball
  • Medium trail (50-100) - Creates a comet effect with a visible tail
  • Long trail (200-500) - Shows most of the spiral, with one end appearing to draw
  • Low repetitions (1-5) - Fast movement, energetic animation
  • High repetitions (50-100) - Slow, smooth movement, hypnotic effect

Export Format

WAV Audio

Oscilloplot exports stereo 16-bit PCM WAV files where:

  • Left channel: X coordinates (horizontal position)
  • Right channel: Y coordinates (vertical position)

Connecting to an Oscilloscope

Step-by-Step Setup

  1. Set oscilloscope to XY mode
    Also called X-Y, dual channel, or Lissajous mode. Consult your oscilloscope manual for the specific setting.
  2. Connect audio output
    • Left channel → X input (Channel 1)
    • Right channel → Y input (Channel 2)
    Use a stereo 3.5mm to dual RCA/BNC adapter cable.
  3. Adjust oscilloscope settings
    • Set time/div to external or X-Y mode
    • Adjust gain on both channels equally
    • Set coupling to AC or DC depending on your pattern
  4. Start playback in Oscilloplot
    Click "Play Audio" to begin sending the signal.
  5. Fine-tune display
    Adjust oscilloscope brightness, focus, and position controls for best visibility.

Recommended Audio Settings

SettingRecommended Value
Sample Rate100-200 kHz (1000 Hz × 100-200 multiplier)
Duration1-2 seconds for smooth looping
Pattern Repeats100-500 for animated effects
Make sure your audio output volume is set to an appropriate level before connecting to your oscilloscope. Very high volumes could potentially damage sensitive equipment.

Troubleshooting

No Audio Output

  • Check that your audio device is selected in system settings
  • Verify the sample rate is supported by your sound card
  • Try lowering the playback multiplier to reduce the actual sample rate
  • Check system volume and make sure audio is not muted

Laggy Preview

  • Use the optimized version (oscilloscope_gui_optimized.py)
  • Reduce Window Size in Live Preview settings to 2,000-3,000 samples
  • Lower the FPS setting to 15-20
  • Disable Live Preview entirely during audio generation

Image Converter Errors

  • Ensure OpenCV is installed: pip install opencv-python
  • Try different processing methods (Simple, Bilateral, DoG)
  • Use the --edit flag to adjust parameters interactively
  • Check that the image file exists and is readable

Pattern Looks Wrong on Oscilloscope

  • Check X/Y channel connections aren't swapped
  • Ensure both channels have the same gain setting
  • Verify oscilloscope is in XY mode, not time-base mode
  • Try adjusting the audio output volume

Effects Not Visible

  • Click "Apply & Generate" after changing effect settings
  • Ensure the effect checkbox is enabled
  • Try increasing effect parameters (amplitude, depth, etc.)
  • Make sure Pattern Repeats is high enough for animated effects

Application Won't Start

  • Verify Python 3.8+ is installed: python --version
  • Check all dependencies are installed: pip install -r requirements.txt
  • Try running from command line to see error messages
  • Make sure you're in the correct directory
